# NetworkExam_Hyunwoo
언리얼 네트워크 과제 프로젝트입니다.
1. 로비맵
- 시작맵으로, 컨트롤은 안되고 UI만 뜬다.
- IP 입력창 / 생성 / 접속 버튼

2. 입장 후
- 입장 대기 상태
- 인원 입장해서 인원수 2명이면 시작 버튼 활성화
  
3. 게임 시작
- 시작 버튼 누르면 3초 후 게임 시작(생성 시작)
- 점수는 호스트 좌측, 클라이언트 우측에 표기
- 뒤에 ProgressBar로 점수 / 총점 수치값으로 게이지 표기
- 아이템 획득 시 7점 획득
- 시간 종료되면 결과화면 표기 및 점수 미반영

4. 결과화면
- 승리 / 패배 / 동점에 따라 텍스트 변경
- 승리 시 본인 점수 녹색 표기
- 패배 시 본인 점수 빨강 표기
- 동점 시 흰색 표기
